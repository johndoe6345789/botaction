# viewer_hotspots.js

## Overview

This file contains **CSS module class mappings** - the compiled CSS class name mappings generated by CSS Modules. These map human-readable class names to unique hashed identifiers for style isolation.

## File Information

- **Status**: Active webpack bundle
- **Size**: ~28KB (minified)
- **Type**: CSS Modules mapping
- **Build Tool**: Webpack CSS Modules loader

## CSS Modules System

### How It Works

```javascript
// Original CSS (button.module.css)
.button {
  padding: 10px 20px;
}
.button--primary {
  background: blue;
}

// Compiled output
{
  "button": "button_a1b2c3",
  "button--primary": "button--primary_d4e5f6"
}

// Usage in React
import styles from './button.module.css';
<button className={styles.button}>Click</button>
// Renders: <button class="button_a1b2c3">
```

## Usage in Components

### React with CSS Modules

```jsx
import styles from './ModelCard.module.css';

function ModelCard({ model, featured }) {
  return (
    <div className={`${styles['model-card']} ${featured ? styles['model-card--featured'] : ''}`}>
      <div className={styles['model-card__thumbnail']}>
        <img src={model.thumbnail} alt={model.name} />
      </div>
      <h3 className={styles['model-card__title']}>{model.name}</h3>
      <span className={styles['model-card__author']}>{model.author}</span>
    </div>
  );
}
```

### classnames Library

```jsx
import cn from 'classnames';
import styles from './Button.module.css';

function Button({ variant, size, children }) {
  const className = cn(
    styles.button,
    styles[`button--${variant}`],
    styles[`button--${size}`]
  );
  
  return <button className={className}>{children}</button>;
}
```

### Dynamic Classes

```jsx
function StatusBadge({ status }) {
  const statusClasses = {
    success: styles['badge--success'],
    warning: styles['badge--warning'],
    error: styles['badge--error']
  };
  
  return (
    <span className={`${styles.badge} ${statusClasses[status]}`}>
      {status}
    </span>
  );
}
```

## Webpack Configuration

```javascript
// webpack.config.js
module.exports = {
  module: {
    rules: [
      {
        test: /\.module\.css$/,
        use: [
          'style-loader',
          {
            loader: 'css-loader',
            options: {
              modules: {
                // Generate unique class names
                localIdentName: '[name]__[local]_[hash:base64:5]',
                // Export class mappings
                exportLocalsConvention: 'camelCase'
              }
            }
          }
        ]
      }
    ]
  }
};
```

## Benefits

1. **Scoped Styles** - No global CSS conflicts
2. **Explicit Dependencies** - Import what you use
3. **Dead Code Elimination** - Unused styles not bundled
4. **Predictable Names** - Unique hashed identifiers
5. **Composition** - Compose styles from other modules

## Composition Example

```css
/* base.module.css */
.text {
  font-family: 'Inter', sans-serif;
}

/* heading.module.css */
.heading {
  composes: text from './base.module.css';
  font-weight: bold;
}
```
